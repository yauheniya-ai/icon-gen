Parameters

Parameter Type Description
icon_name str Iconify icon identifier (e.g., "mdi:code")
direct_url str Direct URL to an image (e.g., "https://upload.wikimedia.org/wikipedia/commons/f/f6/Builder_icon_hicolor.png")
local_file str Path to local image file (e.g., "input/html_css_js.jpeg")
output_name str Output filename (without extension)
format str Output format: "svg", "png", "webp", "ico"
size int Size in pixels (default: 256)
color str or tuple Icon color (solid: "red", gradient: ("red", "blue"))
direction str Icon gradient direction: "horizontal", "vertical", "diagonal"
bg_color str or tuple Background color (default: None)
bg_direction str Background gradient direction: "horizontal", "vertical", "diagonal"
border_radius int Corner radius (use size/2 for circle) (default: 0)
outline_width int Outline width in pixels (default: 0)
outline_color str Outline color
animation str or dict SVG-native animation preset "spin", "pulse", "flip-h", "flip-v". Append duration with ":2s" (e.g. "spin:4s") or pass dict like {"type":"pulse","dur":"1s"}.
