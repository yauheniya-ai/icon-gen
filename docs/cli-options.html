Global Options

Option Description Example
--help Show help message and exit icon-gen-ai --help
--version Show the CLI version and exit icon-gen-ai --version

Generate Command Options

icon-gen-ai generate [ICON] [OPTIONS]

Option Description Example
-i, --input Local file path or direct URL -i input/icon.png
-o, --output Output file path -o icon.svg
--size Size in pixels --size 256
--color Icon color: solid 'black' or gradient '(white,lime)' --color blue
--direction Gradient direction for the icon color (horizontal, vertical, diagonal) --direction diagonal
--bg-color Background color: solid 'black' or gradient '(white,lime)' --bg-color '(red,blue)'
--bg-direction Gradient direction for the background color (horizontal, vertical, diagonal) --bg-direction vertical
--border-radius Corner radius (0=square, size/2=circle) --border-radius 128
--outline-width Outline width in pixels --outline-width 4
--outline-color Outline color --outline-color black
--animation Apply an SVG-native animation preset (spin, pulse, flip-h, flip-v). Append duration like ":2s" (e.g. spin:4s). --animation "pulse:1s"

Search Command Options

icon-gen-ai search QUERY [OPTIONS]

Option Description Example
-n, --count Number of icons to return -n 5
-g, --generate Automatically generate icons from search results --generate
--style Optional design style for AI-generated icons --style modern
--project-type Optional project context for AI-generated icons --project-type "web app"

Providers Command

icon-gen-ai providers

Shows installed AI providers and the currently active provider.
